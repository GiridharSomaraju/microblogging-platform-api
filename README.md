# Microblog Platform API

## Features

- User Registration
- User Login (JWT)
- Create Tweet
- Update Tweet
- Delete Tweet
- Like / Unlike Tweet
- Reply to Tweet
- Delete Reply
- Follow / Unfollow Users
- Get Followers
- Get Following
- Get My Profile
- Get User Profile
- Search Users
- Feed API
- Like Count in Feed

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- JWT
- bcrypt
